Briefly info

Release data: 19 November, 1997
Platform: MS-DOS

Description

At the beginning of the game you have to set the highest number which could be found. After that the computer sets a number by chance in the given area and you have to find that number. You have 200 attempts to find the correct number.
A simple 'find my number' game.
