Cheats for AblazeBall

You could start the game with this parameters:

Parameter This happens
-cheats The program must be started with this parameter to activate the cheats
-invulnerable (F2) You couldn't die
-unlimited_thrust (F3) Your thrust engine is unlimited
-unlimited_terraform (F4) You could manipulate the terrain all the time
-unlimited_sparks (F5) You receive unlimited sparks
-low_graviation (F6) Activates low gravitation
-no_water (F7) Disables/enables the water
-no_fog (F8) Disables/enables the fog
-speed_control (F9 +/-) You could control the game speed by self... slow-motion-effect possible...
F10 win game
